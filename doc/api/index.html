API Documentation

Concerts

  • creates a concert given valid data
  • deletes a concert given an existing ID
  • lists all concerts
  • returns an error when creating a concert with invalid data
  • returns an error when deleting a concert with an unknown ID
  • returns an error when showing a concert with an unknown ID
  • returns an error when updating a concert with an unknown ID
  • shows a concert given an existing ID
  • updates a concert given an existing ID
